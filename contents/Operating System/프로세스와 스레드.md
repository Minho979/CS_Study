# 프로세스와 스레드
초기 컴퓨터는 하나의 프로그램이 프로세서를 독점하여 프로세스를 통한 관리가 필요하지 않았으나 다중 프로그래밍에서 여러 프로그램이 자원을 공유하기에 프로그램들을 제어하는 방법이 요구되었다.  
이에 프로세스를 통해 프로그램을 제어하고 자원을 관리하기 시작하였고 멀티 테스킹이 가능하게 되었다.
