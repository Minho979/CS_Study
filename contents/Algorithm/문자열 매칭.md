# 문자열 매칭
텍스트 문자열 A[1...n]이 패턴 문자열 P[1...m]을 포함하는지를 알아보고 포함하는 경우 어느 위치에 있는지를 알아내는 알고리즘
- n > m
- 예시) A = "aababacccc", P = "aba"

  매칭이 일어난 위치(매칭된 A의 시작 인덱스): 2, 4 (인덱스의 시작을 1로 설정)
